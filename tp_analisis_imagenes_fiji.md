# Trabajo Práctico: Análisis cuantitativo de imágenes

La imagen es una matriz de números, y lo que vemos en la pantalla es una representación.

1.	Abrir una imagen monocromática y explorar el valor en cada pixel.
2.	Medir el perfil de intensidad en una línea.
3.	Cambiar la LUT, brillo y contraste. 
4.	Repetir los pasos 1 y 2; y comparar.
5.	Discutir con el docente las diferencias entre una imagen de 8-bits, 16-bits y 32-bits

Matemática, Filtros
1.	Abrir una imagen monocromática y explorar el valor en cada pixel.
2.	Generar una máscara con un umbral.
3.	Sumar / Restar / Multiplicar / Dividir por un escalar
4.	Sumar / Restar / Multiplicar / Dividir por la otra imagen
5.	Aplicar un filtro para suavizar
6.	Aplicar un filtro para encontrar bordes.
7.	Filtro pasabanda / Transformada de Fourier

Regiones de Interés
1.	Generar una máscara con una ROI
2.	Usar el ROI manager para seleccionar varias ROIs
3.	Mediar propiedades

Color
1.	Abrir imágenes correspondientes de distintos canales
2.	Combinarlas en una imágen color
3.	Cambiar para cada canal la LUT, brillo y contraste

Preparación para publicación
1.	Agregar una barra de calibración
2.	Agregar una barra de scala
3.	Realizar un Montaje con los 3 canales por separado + la imagen color
